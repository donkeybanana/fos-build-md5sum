> FrogOS Build MD5sum


## Install

```sh
$ npm install --save fos-build-md5sum
```


## Usage

```js
var fos-build-md5sum = require('fos-build-md5sum');

fos-build-md5sum('Rainbow');
```

```sh
$ npm install --global fos-build-md5sum
$ fos-build-md5sum --help
```
